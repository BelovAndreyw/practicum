| Личный кабинет, редактирование профиля, достижения, личный рейтинг | Профиль с личным вкладом, ачивками, статусом и историей активности | `docs/ux-ui-mvp.md`, `prototype/index.html` |
| Создание команды капитаном | Отдельные captain-actions: создание команды, приглашение, управление составом | `docs/ux-ui-mvp.md`, `prototype/index.html` |
| Генерация пригласительных кодов и ссылок | Карточка приглашения и QR/кодовый сценарий в captain workspace | `prototype/index.html` |
| Вступление в команду и поиск команд | Онбординг со сценарием поиска или присоединения по коду | `docs/ux-ui-mvp.md` |
| Страница команды с составом, историей и КРК | Team hub с командным баннером, участниками и разложением КРК | `prototype/index.html` |
| Автоматический расчет КРК | В UI КРК разложен на 60/30/10 для прозрачности логики | `docs/ux-ui-mvp.md`, `prototype/index.html` |
| Таблицы лидеров команд и личные рейтинги | Лидерборд с табами "Команды / Участники", карточки лиг | `prototype/index.html` |
| Распределение по лигам | League-chip и статусы уровня команды | `prototype/index.html` |
| Лента активностей | Отдельный блок с последними действиями команды и потока | `prototype/index.html` |
| Челленджи с возможностью загрузки отчетов | Challenge card и CTA на отправку результата | `prototype/index.html` |
| Создание собственных событий командами | Сценарий captain workspace для запуска встречи/воркшопа | `docs/ux-ui-mvp.md`, `prototype/index.html` |
| Новостная лента | Story-strip и news cards | `prototype/index.html` |
| Биржа знаний | Отдельный экран с заявками, ценой в баллах и статусами сделки | `prototype/index.html` |
| Календарь событий | Календарный блок со списком ближайших активностей | `prototype/index.html` |
| Weekly check-in капитанов | Карточка check-in с фиксированным набором полей | `prototype/index.html` |
| Механизм "Спасение" | Формализованный flow со статусами и подтверждением помощи | `docs/ux-ui-mvp.md`, `prototype/index.html` |
| Анонимное внутрикомандное голосование | Экран peer review с пояснением логики и приватности | `prototype/index.html` |
| Административная панель для организаторов | Desktop-консоль с аналитикой, модерацией и управлением контентом | `prototype/index.html` |
| Время отклика до 2 секунд | Для MVP предложен легкий карточный интерфейс без перегруженного первого экрана | `docs/ux-ui-mvp.md` |
| Безопасность и ролевая модель | Разделение сценариев по ролям, верификация, приватные и чувствительные действия вынесены в отдельные интерфейсы | `docs/ux-ui-mvp.md` |
| Масштабируемость | Модульная навигация и компонентный подход для роста числа потоков и команд | `docs/ux-ui-mvp.md` |

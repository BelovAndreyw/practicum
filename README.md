# Командный зачёт - Project Roadmap

В основной ветке сейчас размещены материалы, связанные с дорожной картой проекта.

## Что внутри

- `docs/project-roadmap.md` - подробная дорожная карта проекта по этапам, ролям, результатам и контрольным точкам.
- `prototype/roadmap.html` - графическая веб-версия roadmap для просмотра в браузере.
- `prototype/roadmap.css` - стили для roadmap-страницы.

## Как открыть

- Для полного текстового плана открой `docs/project-roadmap.md`.
- Для наглядного графического представления открой `prototype/roadmap.html` в браузере.

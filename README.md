# Командный зачёт - UX/UI MVP

Этот репозиторий фиксирует UX/UI-часть MVP для веб-приложения "Командный зачёт". Основной фокус - превратить сложную систему командной учебной игры в понятный, мотивирующий и визуально цельный пользовательский опыт для студентов, капитанов и организаторов.

## Что внутри

- `prototype/index.html` - адаптивный hi-fi прототип ключевых экранов MVP.
- `prototype/styles.css` - визуальная система, сетка, компоненты и анимации.
- `prototype/app.js` - переключение ролей и базовая интерактивность прототипа.
- `docs/ux-ui-mvp.md` - UX/UI-концепция, роли, сценарии, IA и дизайн-решения.
- `docs/mvp-coverage.md` - покрытие требований ТЗ с точки зрения UX/UI.

## Как открыть

Открой файл `prototype/index.html` в браузере. Это статический прототип без сборки и зависимостей.

## Что закрывает MVP

- Верификацию и вход по вузовому сценарию.
- Личный кабинет студента с достижениями и вкладом в команду.
- Командный контур: создание, вступление, приглашения, профиль команды.
- Рейтинги, лиги и прозрачную формулу КРК.
- Ленту активностей, челленджи, новости и календарь.
- Биржу знаний, механику "спасения" и weekly check-in.
- Анонимное внутрикомандное оценивание.
- Организаторскую панель с модерацией, аналитикой и управлением активностями.

## Визуальное направление

Прототип опирается на присланные референсы:

- темный графитовый фон;
- теплый оранжевый акцент;
- крупные скругленные карточки;
- метрики, прогресс и лидерборды как центральный визуальный язык;
- мобильная пластика интерфейса, аккуратно адаптированная под веб.

## Зачем это полезно для MVP

Репозиторий можно использовать как основу для:

- защиты UX/UI-части проекта;
- передачи логики экранов разработчику;
- сборки структуры в Figma;
- обсуждения состава MVP внутри команды.
